# debug

| 命令        | 说明                                                         |
| ----------- | ------------------------------------------------------------ |
| achievement | 解锁或撤销一个成就。如果未提供具体成就，则会解锁或撤销所有成就。 |
| act         | 跳转到某一幕（Act）。如果传入整数，则跳转到对应章节；否则会将当前章节替换为指定章节。 |
| afflict     | 对玩家手中的一张卡牌施加指定的负面效果（Affliction）。       |
| ancient     | 打开一个 Ancient 事件，并选择指定选项。                      |
| power       | 给指定索引的目标施加 Power 效果。                            |
| art         | 列出指定类型中缺少美术资源的所有内容。类型包括：affliction、card、enchantment、power、relic。 |
| block       | 为玩家增加格挡；如果指定目标索引，则为目标生物增加格挡（0 为玩家）。 |
| card        | 向某个牌堆生成一张卡牌（默认加入手牌）。卡牌名需使用全大写蛇形命名，例如：BODY_SLAM，而不是 Body Slam。 |
| cloud       | 删除 Steam Cloud 中的所有存档（仅 Steam 版本有效）。         |
| damage      | 对所有敌人造成伤害；如果指定目标索引，则对指定生物造成伤害（0 为玩家）。 |
| die         | 立即死亡。                                                   |
| draw        | 抽取 X 张卡牌。                                              |
| dump        | 将 Model ID 数据库导出到控制台与日志文件。                   |
| enchant     | 为玩家手中的一张卡牌附加指定附魔（Enchantment）。            |
| energy      | 为玩家增加能量。                                             |
| event       | 将玩家跳转到指定事件。                                       |
| fight       | 将玩家跳转到指定战斗遭遇。                                   |
| getlogs     | 收集日志，自动打包为 zip 文件（文件名包含指定 name），并打开该 zip 文件所在目录。 |
| godmode     | 开启无敌模式。                                               |
| gold        | 修改玩家金币数量。Cha-ching!                                 |
| heal        | 为玩家恢复一定量生命值。                                     |
| instant     | 开启 Instant 模式。                                          |
| kill        | 击杀指定目标；如果参数为 `all` 则击杀全部；如果不传参数则击杀第一个目标。 |
| leaderboard | 向排行榜添加分数。选项可为 `upload` 或 `random`。如果为 `random`，会向指定排行榜上传 `<count>` 个随机分数；如果为 `upload`，则会向指定排行榜上传一个 `<score>` 分数。 |
| log         | 设置指定日志类型的日志级别。类型包括：Generic、Network、Actions、GameSync、VisualSync。等级包括：VeryDebug、Load、Debug、Info、Warn、Error。 |
| multiplayer | 打开多人模式菜单；如果第一个参数为 `test`，则打开测试场景。  |
| open        | 在本地操作系统文件浏览器中打开一个常用路径。                 |
| potion      | 添加药水到腰带。药水名需使用全大写蛇形命名，例如：ENTROPIC_BREW，而不是 Entropic Brew。 |
| relic       | 为玩家添加/移除遗物（默认是添加）。                          |
| remove_card | 从手牌或牌组中移除一张卡牌。卡牌名需使用全大写蛇形命名，例如：BODY_SLAM，而不是 Body Slam。 |
| room        | 将玩家跳转到指定房间。                                       |
| sentry      | 测试 Sentry 错误上报。`test` 发送测试消息与异常；`message <text>` 发送自定义消息；`exception` 抛出测试异常；`crash confirm` 触发原生崩溃（会终止游戏）；`status` 显示 Sentry 状态。 |
| stars       | 为玩家增加星星。                                             |
| trailer     | 切换 UI 元素显示/隐藏功能，可通过 `0-9` 与 `+-` 按键控制。   |
| travel      | 允许玩家跳转到地图上的任意房间。                             |
| unlock      | 将所有卡牌/药水/遗物/怪物/事件/时代/进阶等级标记为已发现；参数为 `all` 时解锁全部内容。 |
| upgrade     | 根据手牌位置升级目标卡牌（0 为最左侧）。                     |
| win         | 立即赢得当前战斗。                                           |
| showlog     | 打开日志显示窗口。                                           |
