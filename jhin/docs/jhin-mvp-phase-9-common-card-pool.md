# Jhin MVP 文档 - 阶段 9：普通牌完整实现

## 1. 阶段目标

本阶段目标是完成 35 张普通牌，实现烬在游戏前期的完整体验。

本阶段重点不是稀有爆发上限，而是保证普通牌池本身就能支撑角色在早期形成明确方向，并且不会因为 0 费射击、装填收益或普通奖励分布失衡而破坏第四枪节奏。

## 2. 本阶段交付内容

必须完整实现设计文档中的全部普通牌：

- 普通攻击牌 15 张
- 普通技能牌 11 张
- 普通能力牌 9 张

合计：35 张普通牌。

## 3. 范围定义

1. `普通射击`
2. `精准一枪`
3. `低语点射`
4. `瞄准射击`
5. `穿透弹`

8. `收尾`
9. `侧身开火`
10. `快速拔枪`
11. `试演一枪`
12. `枪焰擦伤`
13. `幕间点杀`
14. `余兴弹`
15. `低声威胁`

## 3.2 普通技能牌 11 张

## 4. 功能目标

普通牌池完成后，必须支撑完整的前期玩法建立。

最低目标：

- 初始卡组能自然过渡到普通奖励构筑。
- 玩家在第一层就能从普通奖励中看出 Build 方向。
- 普通牌之间存在清晰但不过度失控的联动。

## 5. 重点验证方向

本阶段必须围绕以下四项重点验证。

## 5.1 初始卡组强度是否失衡

需要验证：

- 初始卡组是否明显过弱，导致前期生存困难。
- 初始卡组是否明显过强，导致普通奖励缺乏吸引力。

重点观察：

- `普通射击` 与 `优雅走位` 是否足够支撑基础回合。
- `装填` 是否过强地抹平了弹匣规划成本。
- `致命华彩` 是否过早提供过高爆发。

## 5.2 普通奖励是否能稳定形成 Build 方向

需要验证：

- 普通奖励是否能自然导向至少三条早期路线。
- 同一条路线是否有足够的普通牌相互支撑，而不是只靠少数关键牌。

必须能支撑的早期路线：

- 第四枪爆发
- 标记狙杀
- 装填循环

## 5.3 0 费射击是否导致华彩过频

需要验证：

- `低语点射`
- `快速拔枪`

等低费或 0 费射击是否导致第四枪过于轻松实现。

重点观察：

- 普通战斗中是否过于容易连续触发华彩。
- 0 费射击是否压缩了其他中速射击牌的选择空间。
- 是否出现“只拿 0 费射击就能稳定爆发”的单一策略。

## 5.4 装填牌是否削弱第四枪规划感

需要验证：

- `装填`
- `冷静装填`
- 与装填相关的能力牌

是否让玩家可以无脑重置节奏，而不再需要规划何时打空弹匣。

重点观察：

- 装填是否只是容错工具，还是已经变成高收益主循环。
- 玩家是否仍需要考虑“把哪张牌作为第四枪”。
- 装填是否让空弹风险变得几乎不存在。

## 6. 早期 Build 支撑要求

普通牌池必须至少能支撑以下三条路线在前期成型。

## 6.1 第四枪爆发

普通牌侧重点：

- `普通射击`
- `精准一枪`
- `低语点射`
- `穿透弹`
- `试演一枪`
- `余兴弹`

目标：

- 玩家能围绕第四枪规划伤害峰值。
- 普通牌本身就能让第四枪有明显收益差。

## 6.2 标记狙杀

普通牌侧重点：

- `瞄准射击`
- `布置舞台`
- `观察弱点`
- `低声威胁`
- `致命华彩`

目标：

- 玩家在普通奖励阶段就能识别“先铺标记再引爆”的路线。
- 标记牌数量足以支撑前期连续构筑。

## 6.3 装填循环

普通牌侧重点：

- `装填`
- `冷静装填`
- `精密计算`
- `后撤步`
- `演出计划`
- `枪械保养`

目标：

- 玩家能通过装填与抽牌构成稳定回合。
- 但不能因此完全抹掉第四枪节奏成本。

## 7. 单卡实现要求

35 张普通牌全部需要满足：

1. 基础效果正确。
2. 升级效果正确。
3. 文本描述与实际行为一致。
4. 类型、稀有度、目标、费用配置正确。
5. 统一接入已有系统：
   - 射击
   - 华彩
   - 标记
   - 莲华陷阱
   - 装填
6. 卡图默认接入 `JhinMod\Images\card_placeholder.png`。

## 8. 普通牌池平衡观察项

本阶段必须持续关注以下问题：

1. 是否存在明显优于其他选择的普通牌。
2. 是否存在几乎不会被选择的普通牌。
3. 普通能力牌是否太慢，导致早期完全不想拿。
4. 普通技能牌是否过于保守，导致攻击流一边倒更优。
5. 普通攻击牌是否过多挤压技能和能力的构筑空间。

## 9. 建议实现顺序

建议按以下顺序推进，而不是一次性平铺 35 张：

1. 先完成已在 MVP 20 张中的普通牌。
2. 再补第四枪爆发方向的普通攻击牌。
3. 再补标记方向普通牌。
4. 再补装填循环方向普通牌。
5. 最后补普通能力牌，并统一回归前期节奏平衡。

这样可以降低联调压力，并更早发现方向失衡问题。

## 10. 非目标

以下内容不属于本阶段：

- 罕见牌与稀有牌完整实现
- 灵魂卡实现
- 全部遗物与药水完整实现
- 最终平衡定稿
- 单元测试

说明：

- 本阶段重点是把普通牌池做完整并可验证，不要求一次把全角色内容做完。

## 11. 验收标准

本阶段完成后，必须满足以下验收条件：

1. 普通牌池可以支持至少三种早期路线：
   - 第四枪爆发
   - 标记狙杀
   - 装填循环
2. 普通战斗平均每场华彩约 1 到 3 次。

建议补充手动验证场景：

1. 用仅包含初始卡组与普通奖励的流程进行前期战斗，验证生存与输出是否都可接受。
2. 连续多场普通奖励选择后，应能较稳定形成第四枪、标记或装填其中一条路线。
3. 使用多张 0 费射击后，华彩频率不应明显超出预期区间。
4. 多次使用装填牌后，玩家仍应需要规划何时打出第四枪。
5. 普通能力牌在前期不应全部成为“看到就跳过”的低吸引力选项。

## 12. 完成定义

当以下条件全部满足时，可判定“阶段 9 完成”：

- 35 张普通牌已全部可用。
- 普通牌池能独立支撑前期战斗与早期构筑方向。
- 第四枪、标记、装填三条前期主路线都已具备稳定入口。
- 华彩频率处于健康区间，没有明显因 0 费牌或装填牌导致失控。
- 普通奖励已具备“看得出方向、拿得到联动、打得出节奏”的基础质量。

## 13. 下一阶段接口预留

为了减少后续返工，本阶段应预留以下扩展点：

- 普通牌与罕见牌的跨稀有度联动入口
- 更细粒度的战斗统计记录入口
- 普通牌平衡回调与数值热修正入口
- 后续事件、遗物、药水强化普通牌的扩展点

这部分只要求结构可扩展，不要求本阶段实现完整内容。
