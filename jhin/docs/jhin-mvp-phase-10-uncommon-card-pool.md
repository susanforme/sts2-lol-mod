# Jhin MVP 文档 - 阶段 10：罕见牌完整实现

## 1. 阶段目标

本阶段目标是补齐烬的 35 张罕见牌，让主要 Build 在中期具备明确核心牌和稳定成型路径。

本阶段重点不是最终上限，而是让第四枪爆发、标记狙杀、莲华陷阱、谢幕终结这几条核心路线真正进入“围绕关键牌构筑”的状态，同时避免过早出现无限或一回合无脑展开。

## 2. 本阶段交付内容

必须完整实现设计文档中的全部罕见牌：

- 罕见攻击牌 15 张
- 罕见技能牌 11 张
- 罕见能力牌 9 张

合计：35 张罕见牌。

## 4. 功能目标

罕见牌池完成后，必须让角色在中期明显进入“Build 成型阶段”。

最低目标：

- 罕见牌能显著改变出牌顺序、资源规划和终结方式。
- 罕见牌不是简单数值提高，而是能强化一条构筑方向。
- 普通牌提供路线入口，罕见牌提供路线核心。

## 5. 重点 Build 支撑要求

本阶段必须重点围绕以下四条 Build 实现与验证。

## 5.1 第四枪爆发流

核心牌：

- `第四乐章`
- `第三声枪响`
- `完美换弹`
- `华彩节奏`

目标：

- 强化“为了第四枪主动规划前三发”的节奏。
- 让玩家有理由围绕华彩、能量与装填建立一套爆发回合。

重点验证：

- `第三声枪响` 到 `第四乐章` 的衔接是否自然。
- `完美换弹` 是否提供强力节奏重置，但不会直接抹掉规划成本。
- `华彩节奏` 是否让华彩收益显著提升，但不会立刻无限展开。

## 5.2 标记狙杀流

核心牌：

- `死亡预告`
- `镜头锁定`
- `远距狙击`
- `名作诞生`

目标：

- 让标记从“普通牌辅助增伤”提升为一条明确构筑主线。
- 强化单体爆发与击杀滚雪球能力。

重点验证：

- 高层标记是否能形成显著但可控的爆发。
- `镜头锁定` 的“不移除标记”是否会造成异常叠伤。
- `名作诞生` 的击杀奖励是否足够吸引，但不会让多敌战失控。

## 5.3 莲华陷阱流

核心牌：

- `盛大布景`
- `暗藏机关`
- `引爆莲华`
- `莲华工坊`

目标：

- 让陷阱从单纯防守工具提升为一条多敌战主线。
- 让玩家能够选择“等待敌人攻击触发”或“主动引爆收尾”两种节奏。

重点验证：

- `盛大布景` 是否足以支撑多敌战起手。
- `暗藏机关` 是否增强单体叠层效率。
- `引爆莲华` 是否能提供明确爆发窗口。
- `莲华工坊` 是否让标记与陷阱两条路线形成自然交叉。

## 5.4 谢幕终结流

核心牌：

- `华丽处刑`
- `完美谢场`
- `死亡是艺术`

目标：

- 让残局收割和击杀收益成为独立 Build 方向。
- 强化“压低血线后终结”的玩法感。

重点验证：

- `华丽处刑` 是否能稳定承担中高费收割职责。
- `完美谢场` 是否为华彩后防守回合提供稳定支撑。
- `死亡是艺术` 是否能创造击杀连锁收益，但不会直接形成无限。

## 6. 单卡实现要求

35 张罕见牌全部需要满足：

1. 基础效果正确。
2. 升级效果正确。
3. 文本描述与实际行为一致。
4. 费用、类型、目标、关键字配置正确。
5. 统一接入已有系统：
   - 射击
   - 华彩
   - 标记
   - 莲华陷阱
   - 装填
   - 击杀收益
6. 卡图默认接入 `JhinMod\Images\card_placeholder.png`。

## 7. 中期构筑平衡观察项

本阶段必须持续关注以下问题：

1. 罕见牌是否真正提供了 Build 核心，而不只是高数值卡。
2. 某一条 Build 是否明显强于其他路线。
3. 装填与华彩收益叠加后是否过快形成循环失控。
4. 标记高层叠加是否让单体伤害过早超模。
5. 陷阱引爆是否在多敌战里过强或过弱。
6. 击杀收益牌是否在精英战中产生明显滚雪球失衡。

## 8. 关键高风险组合回归

本阶段必须重点回归下列组合：

1. `华彩节奏 + 第四乐章 + 完美换弹`
2. `第三声枪响 + 第四乐章`
3. `死亡预告 + 镜头锁定 + 远距狙击`
4. `名作诞生 + 标记击杀`
5. `盛大布景 + 暗藏机关 + 引爆莲华`
6. `莲华工坊 + 标记来源牌`
7. `华丽处刑 + 死亡是艺术`

目标：

- 这些组合应当“显著强”，但不能直接越过资源约束，形成稳定无限或毫无代价的连锁。

## 9. 建议实现顺序

建议按 Build 分组推进，而不是按卡牌编号顺序平铺：

1. 先完成第四枪爆发流核心牌。
2. 再完成标记狙杀流核心牌。
3. 再完成莲华陷阱流核心牌。
4. 再完成谢幕终结流核心牌。
5. 最后补其余罕见牌并统一回归中期战斗节奏。

这样更容易边实现边验证一条路线是否真正成立。

## 10. 非目标

以下内容不属于本阶段：

- 稀有牌完整实现
- 灵魂卡实现
- 全部遗物与药水完整实现
- 最终全量平衡定稿
- 单元测试

说明：

- 本阶段重点是中期 Build 成型，而不是终局上限。

## 11. 验收标准

本阶段完成后，必须满足以下验收条件：

1. 每个 Build 至少有 3 到 5 张关键支持牌。
2. 罕见牌能显著改变打法，但不直接形成无限。
3. 精英战平均每场华彩约 3 到 6 次。

建议补充手动验证场景：

1. 第四枪爆发流中，玩家应能围绕 `第三声枪响`、`第四乐章`、`完美换弹` 打出规划性的爆发回合。
2. 标记狙杀流中，`死亡预告 + 远距狙击` 应形成明显单体斩杀能力。
3. 莲华陷阱流中，`盛大布景 + 引爆莲华` 应在多敌战中产生明确收益。
4. 谢幕终结流中，`华丽处刑` 与 `死亡是艺术` 应让击杀后续行动更有价值。
5. 多轮精英战试玩后，华彩频率应大致落在 3 到 6 次区间，而不是极端过高或过低。

## 12. 完成定义

当以下条件全部满足时，可判定“阶段 10 完成”：

- 35 张罕见牌已全部可用。
- 四条主要 Build 都已具备明确的中期核心牌。
- 罕见牌确实改变了出牌节奏和构筑方向。
- 没有明显依靠单张罕见牌直接打穿资源约束的失控问题。
- 精英战中的华彩频率处于健康区间。

## 13. 下一阶段接口预留

为了减少后续返工，本阶段应预留以下扩展点：

- 稀有牌与终局体系接入入口
- 更高级的 Build 交叉联动扩展点
- 平衡统计与精英战专项记录入口
- 针对无限风险组合的限次或约束入口

这部分只要求结构可扩展，不要求本阶段实现完整内容。
